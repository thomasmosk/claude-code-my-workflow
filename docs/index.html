My Claude Code Setup

Pedro H. C. Sant'Anna · Emory University
Work in progress. This is not a polished guide for everyone. It's mostly a summary of how I've been using Claude Code for academic work — creating lecture slides, writing R scripts, managing Beamer-to-Quarto workflows. I keep learning new things, and as I do, I keep updating these files. This is just a way for me to share what I've figured out with friends and colleagues.

This is a starter kit for academics using Claude Code with LaTeX/Beamer, R, and Quarto. It contains a multi-agent infrastructure — specialized reviewers, quality gates, adversarial QA loops, and automated workflows — that I extracted and genericized from a PhD course I developed (Econ 730 at Emory, 6 lectures, 800+ slides).

What's in the template

  • 10 specialized agents — proofreader, slide auditor, pedagogy reviewer, R reviewer, TikZ critic, domain reviewer, adversarial QA pair, translator, verifier
  • 13 slash commands — /compile-latex, /proofread, /deploy, /qa-quarto, /translate-to-quarto, and more
  • 11 auto-loaded rules — quality gates, notation consistency, R conventions, verification protocol, replication-first coding
  • Quality scoring — 80/90/95 thresholds for commit, PR, and excellence
  • Adversarial critic-fixer loop — two agents that check each other's work across up to 5 rounds
  • Beamer → Quarto pipeline — 11-phase translation with TikZ-to-SVG and ggplot-to-plotly
  • Plan-first workflow — non-trivial tasks start in plan mode; plans saved to disk survive context compression; never /clear
  • Continuous learning — [LEARN:tag] corrections persist in MEMORY.md across sessions, preventing repeated mistakes
  • Automatic session logs — Claude saves what was done, key decisions, and open questions at the end of every session

Getting started

git clone https://github.com/pedrohcgs/claude-code-my-workflow.git my-project
cd my-project

# 1. Edit CLAUDE.md with your project info
# 2. Fill in .claude/rules/knowledge-base-template.md
# 3. Start Claude Code and test: /compile-latex MyFirstLecture
